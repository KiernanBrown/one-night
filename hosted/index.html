Real Time Project 3 - Prototype 1

My goal for this project is to create an online version of One Night Ultimate Werewolf by Bezier Games, while adjusting and adding roles to make my game different. I played this game a lot with friends in my apartment last year, but since then two of them have transfered to different schools, and there has been no way of playing. Although there are other social deduction games that I have found online, nothing really replicates the fast paced and short game structure that One Night has, so I wanted to do it myself. Click Here for a link to a design doc for this project

This project will be created using canvas for the entire game. For this prototype players are placed in a room on their own and have the ability to create and remove other players in the game. Players can also start chatting by hitting enter, and then send a message by hitting enter again. There are also buttons on the button right that allow the player to sleep or wake up, which shows the planned animation for sleeping and waking up in the night.
